# TaskFlow - Task Manager App

A simple task management application where users can register, log in, and manage their tasks through three stages:

- Todo
- In Progress
- Done


## Features

### Authentication
- User Registration
- User Login
- JWT Authentication

### Task Management
- Create Tasks
- Update Tasks
- Delete Tasks
- Move Tasks between Todo, In Progress and Done

### UI
- Responsive Design
- Loading States
- Error Handling

---
